Project PYTHON FLASK / Scrapping events Montpellier Site and API

- Creer le enviroment : 

```
python -m venv venv
```

- Activer le enviroment : 

```
.\venv\Scripts\activate
```

```
pip install -r requirements.txt
```

- Execute le projet :
```
python run.py
```

- Desactiver le enviroment : 

```
deactivate
```
